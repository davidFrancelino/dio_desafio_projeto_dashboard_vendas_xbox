# Desafio de Projeto DIO - Trilha de Análise de Dados com Excel e Copilot - Programa Santander Coders 2025
Esse dashboard tem por objetivo apresentar uma visão gerencial das vendas de assinaturas do Xbox.
Nele, é possível enxergar o volume financeiro em R$ das assinaturas de 1) EA Play, 2) Minecraft Play e 3) Gamepass, permitindo filtro por tipo de subscrição (Anual, Trimestral ou Mensal).

Para utiliza-ló, basta baixar e abrir o arquivo 'dashboard_vendas_xbox.xlsx'.

O dashboard foi construído com dados fictícios, não há exposição de nenhum dado confidencial da companhia, nem dados pessoais e/ou pessoais e sensíveis.
